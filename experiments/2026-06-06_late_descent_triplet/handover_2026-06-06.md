### Open analytical question (the elongation investigation, didn't finish)

User raised a hypothesis at Step 4: R1 regresses on three folds (7, 9, 13), which the user identified as having **elongated cycles**. The question was whether this hypothesis generalises beyond those three folds, or whether they're isolated anomalies and the regime pattern is overfit narrative.

The investigation went through three operationalisations:

1. **`days_since_peak > 1.3 × cycle_mean_length`** — failed because `cycle_mean_length` is *adaptive* and recalibrates during sustained elongation, so the threshold never triggers.
2. **Descent gradient (cents/day per fold)** — wrong concept. User clarified: gradient (slope) is independent of elongation (calendar). My measurement averaged across the fold, missing localised shallow patches anyway.
3. **Per-fold elongation exposure** using cycle-level identification, real cycles only (≥7d duration), frozen 730-day trailing baseline, weighted by per-day cycle ratio. This is the methodologically right approach.

**Result of attempt 3:** hypothesis does NOT cleanly generalise.

| Run | hard25 r | hard10 r | lated r |
|---|---|---|---|
| R1_ABC | −0.21 | −0.20 | −0.16 |
| A only | −0.25 | −0.19 | −0.29 |
| B only | **−0.37** | **−0.39** | −0.26 |
| C only | +0.16 | +0.05 | +0.02 |

Convention: negative r = more elongation correlates with helping more (negative delta).

Key surprises:
- R1 correlation is *negative*, not positive — slight tendency to help more on elongated folds, opposite of hypothesis.
- B has the strongest negative correlation, contradicting the earlier 3-fold stratified table.
- Fold 3 (elongation 1.35, contains the 2022-06-21 cycle the user flagged as "elongated and a little shallow") is where R1 helps the most (delta_hard25 = -0.24). One huge counterexample driving much of the negative correlation.
- Fold 9 is the strongest regressor (delta = +0.118) but only moderate elongation (1.11) — *not* explained by my elongation metric.

**My read:** some combination of (a) the elongation metric still missing something the user's eye captures (back-to-back moderate cycles?), and (b) folds 7/9/13 may be regressing for *different* reasons — not a shared regime. The user separately noted that gradient is independent of elongation. Possible interaction: **extreme elongation + shallow gradient = bad; extreme elongation + normal gradient = fine.** That would explain why fold 3 (elongated, the user said "a little shallow") helped but fold 7 (elongated, the user said "very shallow") hurt. Speculative; not tested.
